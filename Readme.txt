Whatsapp Automation By Phywhatkit
---------------------------------
(First Open Your WebBrowser And Log in With Your Whatsapp Account)
Step:1
Open Your Compiler(Vs Code,Phycharm)
Step:2
Open Your Terminal And Install this pip install pywhatkit
Step:3
Then Type pywhatkit.sendwhatmsg('phone_no', 'message', time_hour, time_min)
Step:4
Enter Phone with Country Code
Step:5
Enter Your Message in 'Message'
Step:6
Enter Time in 24 format

Enter Minute
